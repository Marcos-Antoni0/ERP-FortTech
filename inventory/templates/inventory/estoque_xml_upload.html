Entrada de Estoque via XML

Envie um arquivo XML (ex.: NFe) para extrair itens de estoque. Os campos esperados são {{ expected_fields|join:', ' }}. Após a leitura, você poderá revisar e editar os valores em uma tabela antes de salvar.

  • O código do produto deve existir no catálogo para aplicar o estoque.
  • A quantidade extraída será somada ao estoque atual de cada item.
  • Se preço ou custo não forem informados no XML, usaremos o valor do produto cadastrado.
Carregue o XML completo da nota fiscal para que os itens sejam pré-carregados.
